# Proyecto: Leyes de la Gestalt - Página Web Interactiva

## Descripción
Esta página web tiene como objetivo representar visualmente las leyes de la Gestalt a través de ejemplos interactivos y animaciones. Se ha buscado que el diseño sea accesible, atractivo y responsivo, aplicando técnicas modernas de UX/UI como el efecto Parallax y un esquema visual coherente.

---

## Leyes de la Gestalt Representadas
La página incluye las siguientes leyes de la Gestalt:

- **Ley de la Proximidad**  
- **Ley de la Semejanza**  
- **Ley de la Continuidad**  
- **Ley del Cierre**  
- **Ley de Figura y Fondo**  

---

## Ejemplos Visuales
Cada ley se ha representado mediante elementos interactivos o visuales:

- **Proximidad:** Agrupación de elementos cercanos para indicar relación.  
- **Semejanza:** Uso de colores y formas similares para asociar elementos.  
- **Continuidad:** Líneas y formas que guían la vista del usuario a través de la página.  
- **Cierre:** Figuras incompletas que el usuario completa visualmente.  
- **Figura y Fondo:** Contraste entre elementos principales y el fondo para resaltar la información.

---

## Efecto Parallax
Se ha aplicado el efecto Parallax en las siguientes secciones:

- **Sección de encabezado:** Fondo que se mueve más lentamente que el contenido principal al hacer scroll.  
- **Sección de ejemplos visuales:** Elementos gráficos con movimiento de profundidad para resaltar interacciones y mantener el interés visual.

---

## Accesibilidad
Se han implementado varias medidas para garantizar la accesibilidad:

- **Contraste adecuado:** Colores seleccionados con suficiente contraste para facilitar la lectura.  
- **Jerarquía de títulos:** Uso correcto de etiquetas `<h1>` a `<h3>` para estructurar la información.  
- **Alternativas a color o imágenes:** Se han añadido textos descriptivos (`alt`) en imágenes y iconos, evitando depender únicamente del color para transmitir información.

---

## Despliegue
La página ha sido desplegada en **Vercel**:

1. Se vinculó el repositorio de GitHub a Vercel.  
2. Se configuró el build con el comando `npm run build` (o según tu proyecto).  
3. Vercel genera automáticamente la URL de la página pública, accesible para todos los usuarios.
